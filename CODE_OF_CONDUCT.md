# Code of Conduct

## Our standard

This project expects respectful, technically focused collaboration.

- Be direct, but do not insult or demean contributors.
- Critique code and decisions, not people.
- Assume good intent, but be precise when calling out risk.
- Keep discussions accessible for contributors working in English or Korean.

## Unacceptable behavior

- Harassment, discrimination, or intimidation
- Personal attacks
- Repeated bad-faith derailment of technical discussions
- Publishing private information without consent

## Enforcement

Maintainers may edit, lock, or remove contributions that violate these standards and may restrict repository access for repeated abuse.
